Lab Manager Light - Rest API

The Lab Manager Light provides a rest api for managing vm, host and config resources.

The rest api is still under developement and currently not usable.

VM Resources

A GET request to api/vm will return a list of all known vms.

A GET request to api/vm/:uuid will return detailed infos to a vm given by an uuid.

A POST request to api/vm will create a vm with the given properties.

A DELETE request to api/vm/:uuid will delete a vm given by an uuid.

Host Resources

A GET request to api/host will return a list of all known hosts.

Config Resources

A GET request to api/configuration will return the current configuration.
